How to clean status bars in framed screenshots and recordings.

Use this workflow when the app capture is good, but the captured status bar makes the final mockup feel unfinished.

5 stepsDevice frames9:41 polish

Step 01

Start with a supported iPhone frame.

Clean Status Bar is a frame-media polish tool. Add the iPhone frame first so the screenshot or recording has a device context.

Screenshot To Add

Frame picker or canvas with the selected iPhone frame.

Step 02

Import the screenshot or screen recording.

Place the media inside the frame and check that the capture fills the screen opening cleanly before styling the canvas.

Screenshot To Add

Raw capture inside the frame before cleanup.

Step 03

Enable Clean Status Bar in frame editing.

Turn on Clean Status Bar to replace messy captured status bar content with a clean Apple-style 9:41 treatment when the media is supported.

Screenshot To Add

Frame edit sheet showing the Clean Status Bar toggle.

Step 04

Preview the cleaned result with captions and background.

Check the full composition, not only the device. The cleaned status bar should support the mockup without pulling attention from the app UI.

Screenshot To Add

Before/after canvas preview with captions visible.

Step 05

Export stills or videos from the same project.

Because the state belongs to the frame, keep exporting from Bezel Studio so the cleaned status bar remains part of the final output.

Screenshot To Add

Export preview showing the cleaned frame media.
