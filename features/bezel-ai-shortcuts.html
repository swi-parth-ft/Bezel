Move from idea to polished output with less manual setup.

BezelAI and Quick Mockups sit on top of the same canvas system, which means the assistant can edit structured project data instead of guessing at a flattened image.

That makes it useful for both direct edits and automation, from phrase changes to saved screenshot recipes you can trigger through Shortcuts.

How to use BezelAI

  • Open AI mode: Turn on BezelAI from a canvas and describe the change you want.
  • Ask for edits: Request layout adjustments, copy changes, motion ideas, or styling updates in natural language.
  • Review the result: Let the assistant apply structured changes to the same canvas model.
  • Save quick mockups: Store a setup as a reusable preset for shortcut-driven workflows.
  • Automate the handoff: Use Shortcuts or sharing flows to turn screenshots into ready-made mockups fast.

Best for

  • Fast iteration when a design needs small changes, not a rebuild.
  • Repeatable launch workflows that should start from saved presets.
  • Automation-heavy teams that want screenshots to move through the pipeline faster.
